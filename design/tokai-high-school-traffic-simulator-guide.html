7. Python Live Control with TraCI

import traci
traci.start(["sumo-gui", "-c", "bergvliet.sumocfg"])

for step in range(3600):          # simulate 1-hour peak
    traci.simulationStep()
    # Example: monitor queue at Main Road / Dreyersdal
    if step % 60 == 0:
        print("Queue length at Int.4:", traci.lane.getLastStepVehicleNumber("..."))

traci.close()

5. Core Mathematical Methodologies

  • Car-following physics: Krauss (default), IDM, Gipps, Wiedemann (all native in SUMO)
  • Gap acceptance & yielding: Built-in priority rules (matches TIA stop-controlled intersections)
  • Lane changing: MOBIL model
  • Route choice: DUAROUTER / A* / dynamic rerouting → natural rat-runs
  • Demand modelling: Time-dependent O-D matrices using exact TIA percentages
  • Metrics: Delays, queues, v/c ratios – directly comparable to Sidra/Traffix
