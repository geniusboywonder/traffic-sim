---
name: Update tasks and subtasks in chat as work progresses
description: Keep the task/subtask list visibly updated in chat as each item is completed
type: feedback
---

As work progresses, update the task and subtask list in the chat — marking items complete (✅) as they are done, and showing what's currently in progress.

**Why:** User wants to track progress in real time without having to ask. Silent execution feels like a black box.

**How to apply:** After posting the initial plan, re-post or update the checklist in chat each time a task or subtask completes. Use ✅ for done, ⏳ for in progress, and ☐ for pending.
